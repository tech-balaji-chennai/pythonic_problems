Reverse A String Without Affecting Special Characters - 01 - Problem Statement

1) Description:
Implement the function void reverse(char str[]) which takes one argument.

str - Input String

Your task is to reverse the individual words of the given string without affecting special characters.

2) Specification (I/O Format):

i) Input Format:
A single string str

ii) Output Format:
Display the string for the given input

3) Example (Sample Test Cases):

i) Sample Input 1:
 a*bcd,e$
ii) Sample Output 1:
 e*dcb,a$

i) Sample Input 2:
 he$ll@o!
ii) Sample Output 2:
 ol$le@h!
