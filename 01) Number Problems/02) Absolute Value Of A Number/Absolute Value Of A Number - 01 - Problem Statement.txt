Absolute Value Of A Number - 01 - Problem Statement

1) Description:
You will be given a number 'num'. Your task is to print the absolute value of 'num'

2) Specification (I/O Format):

i) Input Format:
A single integer num

ii) Output Format:
Display the non-negative integer for the given input

3) Example (Sample Test Cases):

i) Sample Input 1:
 6
ii) Sample Output 1:
 6

i) Sample Input 2:
 -3
ii) Sample Output 2:
 3

4) Definition:
Absolute Number is defined as the distance between the given number and the origin (0)
