Odd Or Even Number - 01 - Problem Statement

1) Description:
You will be given a number 'num'. Your task is to find whether the given number is Odd or Even

2) Specification (I/O Format):

i) Input Format:
A single integer num

ii) Output Format:
Display the string for the given input
Display "Odd" if num is not divided by 2
Display "Even" if num is divided by 2

3) Example (Sample Test Cases):

i) Sample Input 1:
 4
ii) Sample Output 1:
 Even

i) Sample Input 2:
 1
ii) Sample Output 2:
 Odd

4) Definition:
Even Number is a number when divided by 2 leaves a remainder  0
Odd Number is a number when divided by 2 leaves a remainder  1
