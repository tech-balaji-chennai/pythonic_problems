Armstrong Number - 01 - Problem Statement

1) Description:
You will be given a number 'num'. Your task is to check whether the given number is armstrong number or not.

2) Specification (I/O Format):

i) Input Format:
A single integer num

ii) Output Format:
Display the string for the given input
Display "Armstrong Number" if num is an armstrong number
Display "Not Armstrong Number" if num is not an armstrong number

3) Example (Sample Test Cases):

i) Sample Input 1:
 153
ii) Sample Output 1:
 Armstrong Number

i) Sample Input 2:
 321
ii) Sample Output 2:
 Not Armstrong Number

4) Definition:
Sum of cubes of individual digits of a given number equals the original number itself.
