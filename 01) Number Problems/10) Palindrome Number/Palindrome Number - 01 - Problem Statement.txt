Palindrome Number - 01 - Problem Statement

1) Description:
You will be given a number 'num'. Your task is to find Palindrome Number

2) Specification (I/O Format):

i) Input Format:
A single integer num

ii) Output Format:
Display the string for the given input
Display "Palindrome" if num is a palindrome
Display "Not Palindrome" if num is not a palindrome

3) Example (Sample Test Cases):

i) Sample Input 1:
 121
ii) Sample Output 1:
 Palindrome

i) Sample Input 2:
 345
ii) Sample Output 2:
 Not Palindrome

4) Definition:
Palindrome Number is the reverse number 'rev' which is equal to the original number 'orig'
