Sum Of Digits - 01 - Problem Statement

1) Description:
You will be given a number 'num'. Your task is to find the Sum Of Digits of 'num'.

2) Specification (I/O Format):

i) Input Format:
A single integer num

ii) Output Format:
Display the integer for the given input

3) Example (Sample Test Cases):

i) Sample Input 1:
 12
ii) Sample Output 1:
 3

i) Sample Input 2:
 345
ii) Sample Output 2:
 12

4) Definition:
To get sum of digits of a number, split the digits from the last position (last digit in each iteration) and add it to sum variable and remove last digit of a number
