Remove Duplicates From The Sorted Array - Problem Statement

Implement the function rmDuplicates(int arr[], int n) which takes two arguments

arr[] - Sorted array
n     - Size of the array

Your task is to remove the duplicates elements from the given sorted array and return the new length.

Note: You should write an in-place algorithm to implement this task (remove duplicates without using additional memory).

Example:

input 1: 
n = 7
arr[] = {1,1,1,3,3,5,5}
output 1: 
 3

input 2: 
n = 5
arr[] = {10,20,30,40,50}
output 2: 
5
